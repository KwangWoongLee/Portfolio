# Portfolio Server Project

포트폴리오 제작을 위한 C++ 게임 서버 프로젝트 저장소입니다.  
학생 시절부터 개발해 온 IOCP 기반 네트워크 코어를 현재 지식 수준에 맞게 리팩터링하고,  
간단한 멀티 서버 구조(존/월드/상위 서버)와 콘텐츠를 구현하는 것을 목표로 합니다.

---

## 개발 및 테스트 환경

이 프로젝트는 아래 환경에서 개발 및 테스트되었습니다.

- OS : Windows 10 / 11 (64-bit)
- CPU : Intel Core i5-12400F
- RAM : 32GB
- IDE : Visual Studio 2022 Community
- 언어 : C++23 (`/std:c++latest`)

---

## 구현 계획 및 상세

1. **네트워크 라이브러리**
   - IOCP 기반 코어 개발  
   - 세션 관리, 비동기 I/O 처리, 워커 스레드 풀 등 기본 네트워크 인프라 구현

2. **Timer 시스템**
   - FSM 기반 콘텐츠 구현의 핵심 인프라  
   - 주기/일회성 타이머 관리, 콜백 실행, 콘텐츠 스케줄링에 활용

3. **역할별 서버군 개발**
   - 존(Zone), 월드(World), 상위(예: Universe) 서버 개발  
   - 서버 간 통신 환경 구축 및 이벤트 전파 구조 설계

4. **콘텐츠 핵심 동작 구현**
   - 타이머와 FSM을 활용한 기본 콘텐츠 동작 구현  
   - 예: 공성전(Realm Siege) 시작/종료, 유저 이동 처리 등 최소 핵심 플로우

---

## 향후 확장 아이디어

1. **DB 연동**
   - MSSQL, Redis 연동  
   - 콘텐츠 상태, 유저/길드 정보, 매치/랭킹 데이터 저장 및 조회

2. **웹 서버 활용**
   - 운영툴(관리자 페이지) 연동용 웹 서버  
   - 인증 서버, 매칭 서버 등 HTTP/REST 기반 보조 서버와의 통신

---

## 라이선스 / 참고

- 이 프로젝트는 **개인 포트폴리오 및 학습 목적**의 프로젝트입니다.
- IOCP 네트워크 코어는 학생 시절부터 작성해 온 개인 코드를  
  현재 지식 수준으로 리팩터링한 것이며,  
  **현재 재직 중인 회사의 소스 코드는 포함하지 않습니다.**
